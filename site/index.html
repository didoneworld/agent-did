Open agent identity infrastructure

Identity, access, and governance for AI agents.

Agent DID is an open source, vendor-agnostic control plane for managing AI agent identities, authorization metadata, lifecycle state, and enterprise identity integrations using OIDC, SAML, SCIM, Shared Signals, and W3C Decentralized Identifiers.

What it provides

A practical foundation for agent IAM.

Define trusted agent records, operate a tenant-scoped registry, and connect identity and lifecycle controls to the systems enterprises already use.

DID-backed agent records

Model each agent with a W3C DID, environment, trust level, capabilities, protocol bindings, authorization metadata, and governance endpoints.

Enterprise identity hooks

Integrate OIDC, SAML, bearer sessions, SCIM lifecycle flows, and Shared Signals Framework routes from a single FastAPI service.

Control plane operations

Bootstrap tenants, issue API keys, store agent records, inspect audit events, manage FGA tuples, and deprovision agents.
